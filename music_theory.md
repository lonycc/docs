**电吉他新手须知**

1. 初学者练习应该用失真音色还是使用本音？

练习基本功和技巧时使用本音，乐曲练习时使用失真音色.

2. 从主音开始还是节奏优先？

主音和节奏一样重要, 都要大量练习.

3. 急速的solo弹满整个乐段？

爬格子练习

主和弦/下属和弦/属和弦 重要的三和弦

低音(数字下加一点), 倍低音(加两点)
高音(数字上加一点), 倍高音(加两点)

全音符       (X - - -)
二分音符     (X - )
四分音符     (X)
八分音符     (X一条下划线)
十六分音符    (X两条下划线)
三十二分音符  (X三条下划线)
六十四分音符  (X四条下划线)

全休止符 （ 0 0 0 0 )
二分休止符     (0 0)
四分休止符     (0)
八分休止符     (0一条下划线)
十六分休止符   (0两条下划线)
三十二分休止符 (0三条下划线)

休止符时值 = 停顿时长

2/4  四二拍(四分音符为1拍，每小节2拍)   强, 弱
3/4  四三拍(四分音符为1拍, 每小节3拍)   强, 弱, 弱
4/4  四四拍(四分音符为1拍, 每小节4拍)   强, 弱, 次强, 弱
3/8  八三拍(八分音符为1拍, 每小节3拍)   强, 弱, 弱
6/8  八三拍(八分音符为1拍, 每小节6拍)   强, 弱, 弱, 次强, 弱, 弱
5/4  四五拍(四分音符为1拍, 每小节5拍) 2/4 + 3/4 或 3/4 + 2/4

一小节 = 两条相邻长竖线之间的拍子

单拍子  每小节只有一个强拍, 2/4拍 3/4拍 3/8拍
复拍子  每小节超过一个强拍, 4/4拍 6/8拍

附点音符 增加前面音符时值的一半  X· = XX_ (附点四分音符等于四分音符加八分音符)
复附点音符 X.. = 四分音符+八分音符+十六分音符

附点全音符 X - - - - -
附点二分音符 X - -

连音线 将两个或以上音符练成一个音（三种形式, 同小节/跨小节/两个连音线(长的包含短的)

三连音 将一个音符切分成三等分
八分音符三连音 = 四分音符三等分后的一分
四分音符三连音 = 二分音符三等分后的一分

切分音 X_ X X_ 一个四分音符(切分音符)将两个八分音符切分, 其时值=左右两个八分音符时值之和
X__ X_ X__ 一个八分音符将两个十六分音符切分
切分音符会让强音变弱音, 弱音变强音;(?)
需要注意的是, 被切分音符切分的左右两个音符是为一拍(整体)

C #C bD D #D bE E F #F bG G #G bA A #A bB B
音程 两个音符之间的距离, 单位为度(钢琴为例, 两个音符之间的音程就是两个音符之间[含边界]白键个数)

C到C为纯一度(理论上不存在)

C到D为大二度(2键)

C到F为纯四度(5键)
C到G为纯五度(7键)
C到A为大六度(9键)
C到B为大七度(11键)

#X, 升X, 还原至前一个白键
bX, 降X, 还原至后一个白键

纯X度 + 1键(半音) = 增X度 （ C到#C增一度, C到#F增四度, C到#G增五度)
纯X度 + 2键 = 倍增X度 ()
纯X度 - 1键 = 减X度 ()
纯X度 - 2键 = 倍减X度 ()

大X度 + 1键 = 增X度
大X度 + 2键 = 倍增X度
大X度 - 1键 = 小X度
小X度 - 1键 = 减X度
小X度 - 2键 = 倍减X度

和弦: 三个及以上不同音构成的音;
三和弦: 三个音以三度关系构成的和弦;
根音 + 三音 + 五音
大三和弦 = 大三度 + 小三度
小三和弦 = 小三度 + 大三度
增三和弦 = 大三度 + 大三度
减三和弦 = 小三度 + 小三度

C大三和弦(CEG), 也称C和弦, 记作C
C小三和弦(CbEG), Cm
C减三和弦(CbEbG), Cdim
C增三和弦(CE#G), Caug

最重要的是1/4/5三个三和弦，也就是主和弦, 属和弦, 下属和弦

七和弦, 以C做根音为例
大七和弦 = 大三和弦 + 大三度  CEGB, Cmaj7
小七和弦 = 小三和弦 + 小三度  CbEGbB, Cm7
大小七和弦(属七和弦) = 大三和弦 + 小三度  CEGbB, C7
半减七和弦(减小七和弦) = 小七和弦的五音降半音  CbEbGbB, Cm7-5
减七和弦 = 减三和弦 + 小三度  CbEbGbbB(七音重降半音), Cdim7

九和弦, 以C做根音为例
大九和弦 = 大七和弦 + 小三度  CEGBD, Cmaj9
小九和弦 = 小七和弦 + 大三度  CbEGbBD, Cm9
属大九和弦 = 属大七和弦 + 大三度 CEGbBD, C9  根音到九音音程为大九度
属小九和弦 = 属小七和弦 + 小三度 CEGbBD, C9  根音到九音音程为小九度
Add九和弦 = 三和弦 + 九音  CEGD, Cadd9
六九和弦 = Add九和弦 + 六音  CEGAD, C69   同五声音阶: 宫商角徵羽
属七降九和弦 = 属九和弦的九音降半音  CEGbBbD, C7b9

挂留和弦, 在三和弦中, 一般用于1/5级和弦(C/G), 以C和弦为例; 
挂二和弦: 二度音D替换三度音E  CDG, Csus2
挂四和弦: 四度音F替换三度音E  CFG, Csus4

转位和弦, 在三和弦中, C和弦(CEG)为原位和弦, 转位过程根音三音五音不变, 只是变顺序;
C和弦根音转八度, EGC, 第一转位; 三音转八度, GCE, 第二转位; 
转位和弦的作用, 是在和声进行中
